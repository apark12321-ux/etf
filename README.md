# ETF 인사이트랩

ETF 전문 해설 콘텐츠 사이트입니다.

## 구성

- `index.html`: 메인 페이지
- `articles.html`: ETF 해설글 모음
- `database.html`: ETF 비교 DB 구조
- `glossary.html`: ETF 용어사전
- `sources.html`: 공식 출처 안내
- `before-buying.html`: ETF 체크리스트
- `about.html`, `contact.html`, `privacy.html`, `terms.html`, `disclaimer.html`: 애드센스 심사용 기본 페이지
- `assets/styles.css`, `assets/app.js`: 디자인 및 검색/필터 기능

## 운영 원칙

- 다른 사이트의 ETF 설명문 복사 금지
- 공식 출처의 사실 데이터만 확인 후 직접 해석
- ETF 추천보다 교육형 콘텐츠 중심
- 투자 유의사항과 출처 페이지 상시 노출

## 배포

GitHub Pages에서 `main` 브랜치의 root를 배포 소스로 설정하면 정적 사이트로 사용할 수 있습니다.
